# react-library-app

## Description
Ce projet a été réalisé dans le cadre académique afin de mettre en pratique les concepts liés au développement logiciel et à l'architecture applicative.

## Technologies utilisées
- Node.js / React / Docker (selon le projet)
- Gestion des dépendances via package.json
- Environnement local

## Installation

```bash
npm install
npm run dev
```

## Objectif pédagogique
Comprendre la structuration d'un projet, la gestion des dépendances, et la mise en place d'une architecture propre.
