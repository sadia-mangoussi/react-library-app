# React Library App

Application front-end développée avec React.

## Objectif

Mettre en pratique la structuration d’une application React avec séparation des composants, services et tests.

## Technologies

- React
- Vite
- JavaScript
- CSS
- Vitest (tests)

## Structure du projet

- src/App.jsx : composant principal
- src/components : composants réutilisables
- src/services : logique d’accès aux données / appels API
- src/styles : styles de l’application
- src/tests : tests unitaires

## Lancement du projet

npm install
npm run dev
